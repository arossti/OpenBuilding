NRC defines three wbLCA purposes — the purpose determines scope, data quality, and reporting depth.

① Informing Design — OPEN
Any professional, any tool. Assessor sets scope. LOD 100–200, secondary data OK. Internal report — no submission. Tools: BEAM, MEC², cove.tool, EcoDesigner★.
② Meeting Requirements — CONDITIONAL
Program/policy sets system boundary, modules, and data quality. A1–A5 min; A1–C4 preferred. SC/LCA practitioner; report under AR/PE seal for permit. LOD 300+ or CCA Class B min. Tools: MEC², One Click LCA, Athena IE, Tally.
Verification optional unless program requires (Vancouver EC: third-party required).
③ Performance Declaration — CERTIFIED
Public declaration. ISO 14044 Cl.5.2. Full A1–C4 + Module D (separate). All EN 15978 indicators or INA/MNA with justification. Sensitivity analysis required. TRACI v2.1 (NRC default). Benchmarks per ISO 21678:2020. CCA Class A / LOD 350–400 minimum. Third-party verification required for comparative assertions.
Data accuracy tiers (NRC Table 10.8):
Preliminary: LOD 100–200 ≈ CCA Class D/C ≈ ±30–50%
Design Dev.: LOD 300 ≈ CCA Class B ≈ ±15–25%
Construction Docs: LOD 350–400 ≈ CCA Class A ≈ ±5–15%
⚠ Performance Declarations require CCA Class A minimum.

Per NRC-funded Priopta Tool Database Comparison Report (2025): same building, same quantities → different results across tools. Understanding why matters for compliance, benchmarking, and tool selection.

⚠ Inter-tool variability: Two root causes identified in the NRC comparison study:
A1–A3 — EPD-based tools (One Click LCA, Tally, EC3) use publisher EPD data; LCI-based tools (Athena IE, BEAM, MEC²) use background database values. Different data source = different A1–A3 results for identical materials.
A4–C — Default assumptions for transport (A4), construction waste (A5), service life/replacements (B4), and end-of-life (C1–C4) vary significantly across tools with no current industry coordination.
Do not compare absolute totals across different tools without understanding their default assumptions.
⚠ EC3 is NOT a full wbLCA tool
EC3 calculates A1–A3 product-stage only. It cannot model A4 transport, A5 construction, B-stage replacements, or C end-of-life. EC3 cannot be used alone to demonstrate compliance with Vancouver EC, TGS WLC, federal Standard on Embodied Carbon, or any cradle-to-grave requirement. Use EC3 for EPD procurement comparison and material selection — then use a full wbLCA tool for compliance.

Per NRC Practitioner's Guide (Oct 2024). Applies to federal projects; referenced by Vancouver EC Program, TGS WLC.

Step 1 — Calculate EC Benchmark (ECBM): Absolute intensity × floor area or functionally equivalent baseline.
Step 2 — Calculate EC Limit: ECL = ECBM × f (f = reduction factor, e.g. 0.9 = 10% reduction)
Step 3 — Calculate EC of Proposed Design (ECP) via wbLCA per EN 15978.
Step 4 — Determine compliance: ECP ≤ ECL ✓
Two Compliance Pathways
A — Absolute Intensity Limit
Project authority sets EC intensity (kgCO2e/m²).
ECBM = ECI × BFAP (or GFA)
BFA = GFA + attached garages. Both BFA and GFA must be reported.
B — Functionally Equivalent Baseline
Build a "typical practice" baseline with same software tool.
Baseline must match proposed design: thermal, structural, and geometric equivalence (±3% any axis, ±1% GFA).
Two sub-approaches: (i) early design iteration; (ii) modify proposed design to common practice.

⚠ XPS & ccSPF insulation: CLF Baseline values blend HFC+HFO products — do NOT use for compliance. Use product-specific HFO-blowing-agent EPD per Canadian HFC regulations. SPFA industry-wide EPD (2018, HFO) for closed-cell spray foam.

Compliance submissions: Class B minimum (Design Estimate — construction documents or BIM equivalent).
Early design/informing: Class D/C (Indicative/Schematic) acceptable.
Cut-off: excluded materials ≤ 1% of total mass of required elements.

③ Manual calculations outside tool (Excel)

Per NRC Practitioner's Guide s.3.3 + Table 4. OmniClass Table 21 hierarchy. If optional elements included, scope must be same in baseline.

✅ Required Scope
A — Substructure
Foundations (wall/column/special/raft/piles), subgrade enclosure walls (all thermal/moisture/fire layers), slabs-on-grade, perimeter insulation, vapour retarder, waterproofing.
B10 — Superstructure
Above + below-grade floors, columns, beams, interior shear walls, balconies, roof structure, canopies, stairs. Fire-rated gypsum / concrete topping on mass timber = required.
B20 — Exterior Vertical Enclosures
Exterior walls (incl. all thermal/moisture/acoustic/fire protection layers), exterior windows (operating + fixed + window wall), exterior entrance and utility doors.
B30 — Exterior Horizontal Enclosures
Roofing (steep + low-slope + canopy incl. insulation + membrane), roof windows + skylights.
🔶 Optional Scope
C — Interiors: Interior Construction (partitions, interior windows/doors, raised floors, suspended ceilings) + Interior Finishes (wall, floor, ceiling finishes).
D — Services: Conveying (elevators), Plumbing, HVAC, Fire Protection, Electrical (incl. PV), Communications. Use TM65/TM65LA methodology for MEP EC where included.
G — Site Work: Excavation, hardscaping (parking lots, plazas, walkways), landscaping.
❌ Excluded
Radon/methane mitigation, dewatering, excavation support (tiebacks, cofferdams), soil treatment, refrigerant emissions, data/software (intangible), land use change.
Default materials (Practitioner's Guide Table 1):
Structure: steel-reinforced concrete, subgrade insulation = XPS (HFO EPD required).
Exterior walls: steel-framed (residential), aluminum curtain wall (commercial storefront), aluminum window wall (residential 7+ storeys).
Roof: polyiso (conventional) or XPS (inverted).
Rebar: CRSI industry-wide EPD. Steel: CISC or AISC industry-wide EPD. Aluminum: AluQuébec preferred.
Concrete: most regionally specific industry-wide EPD (provincial > Canadian > NA). Default strengths: foundations 25 MPa, walls 35 MPa, slabs 35 MPa, columns 40 MPa.
