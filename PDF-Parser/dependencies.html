4. Version drift + upgrade notes

ItemNotes
Bootstrap Icons BfCA apps standardise on 1.11.3. OBJECTIVE (sibling repo at openbuilding-ca/objective) still uses 1.10.3 — plan to bump on next OBJECTIVE rev.
Bootstrap CSS/JS Loaded by BEAMweb only (modal, dropdown, tab, btn-group). PDF-Parser / Matrix / Database do their UI without Bootstrap; Database viewer is a good reference for how far you can go with raw CSS.
SheetJS / xlsx Matches OBJECTIVE's pinned version 0.18.5. Attaches to window.XLSX. Used by BEAMweb for read-only MCE²/BEAM workbook import. No export planned (one-way convenience per BEAMweb.md §0).
PDF.js PDF-Parser uses 4.9.155 loaded locally from lib/ (not CDN). Keep local to avoid CDN downtime for the core feature.
jsPDF + html2canvas OBJECTIVE had these loaded but disabled in favour of browser-native window.print(). BEAMweb follows the same pattern — print is the export.
D3 + dagre + d3-sankey OBJECTIVE only. Used for Sankey (S16) and dependency graph (S17). BEAMweb may adopt when Phase 9 (calc graph) lands.
